## Minimum Viable Product
A Tumblr clone exclusively for complaining. Built on Rails and Backbone.
Users can:

- [] Create accounts
- [] Create sessions (log in)
- [] Create blogs
- [] Create blog posts
- [] View blogs and posts
- [] Subscribe to blogs
- [] View a feed of subscribed blogs
- [] Search blogs by title
